1. Які послуги ви надаєте?

Telegram боти (від простих команд до інтеграції з API, оплатою та базами даних).

Скрипти для автоматизації (парсинг даних, робота з Excel/Google Таблицями, обробка файлів).

Допрацювання та виправлення готових проектів.

2. Які технології ви використовуєте?

Для ботів: Python (Aiogram, Telebot), Node.js (Telegraf), PHP.

Для скриптів: Python (BeautifulSoup, Selenium), Bash, JavaScript.

Бази даних: SQLite, PostgreSQL, MySQL (якщо потрібно).

3. Як відбувається оплата?

50% перед початком роботи, 50% — після завершення.

Оплата через PayPal, Wise, Revolut, банківський переказ (для України).

Криптовалюта (USDT, BTC) — можна обговорити.

4. Який термін виконання?

Прості боти/скрипти — 1–3 дні.

Складні проекти — до 2 тижнів.

Термін може змінитись, якщо виникають додаткові побажання.

5. Чи можна доробити або виправити помилки після здачі проекту?

Так, перші 7 днів — безкоштовні виправлення (якщо це не новий функціонал).

Після цього — $20–50/год за доопрацювання.

6. Чи можна замовити хостинг для бота?

Так, можу: Налаштувати бота на VPS (DigitalOcean, AWS).

Допомогти з безкоштовним хостингом (Heroku, Replit — якщо підходить). Ціна: від $50 (залежить від сервера).

7. Чи є у вас готові роботи для прикладу?

Так, можу показати: Приклади бота для замовлень, чат-підтримки, парсингу.

Скрипти для автоматизації. (Пишіть — скину посилання або код у приватному порядку).

8. Чому варто обрати саме вас?

Ціна нижча, ніж у середнього фрілансера.

Швидка комунікація (відповідаю протягом 1–2 годин).

Гарантія якості — код завжди коментую, щоб було зрозуміло.
